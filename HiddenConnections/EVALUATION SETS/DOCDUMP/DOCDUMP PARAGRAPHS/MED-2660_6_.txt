INTERVENTION: With each meal, each patient in the trial group received 1 capsule for 3 months, which contained 500 mg turmeric, of which 22.1 mg was the active ingredient curcumin (3 capsules daily). The control group received 3 capsules (1 with each meal) for the same period, which contained starch and were identical in color and size to capsules given to patients in the trial group. MAIN AUTOMATIC MEASURE: Data were analyzed using Statistical Package for the Social Sciences software version 15.0.
